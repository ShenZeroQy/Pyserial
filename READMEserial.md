# 基于Python3、Qt5实现串口助手
1. 安装PyQt5 
 ```python
    pip install  PyQt5 -i https://pypi.tuna.tsinghua.edu.cn/simple
    pip install  pyqt5-tools -i https://pypi.tuna.tsinghua.edu.cn/simple
```
2. 安装串口库
 ```python
    pip install pyserial -i https://pypi.tuna.tsinghua.edu.cn/simple
```
3. 直接运行pyserial.py
